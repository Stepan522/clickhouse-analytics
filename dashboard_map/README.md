# Карта клиентов для выездного мастера в Yandex DataLens

Дашборд доступен по ссылке:
https://datalens.yandex/oj23bloaqglc8

## Описание проекта

Разработан интерактивный dashboard в Yandex DataLens для выездных мастеров и сервисных компаний.

Основная задача решения — повысить эффективность маршрутов и увеличить дополнительные продажи за счет повторных контактов с клиентами в ближайших районах.

Dashboard отображает клиентов на карте с подробными карточками:

- ФИО
- телефон
- комментарии
- адрес
- статус обслуживания
- любая дополнительная информация по клиенту

---

# Бизнес-задача

У выездного мастера часто появляется свободное время между заявками или поездками.

Система помогает быстро определить:

- каких клиентов мастер уже обслуживал рядом;
- кому можно предложить повторное обслуживание;
- кому можно продать дополнительные услуги;
- какие точки находятся по пути текущего маршрута.

## Пример сценария

Мастер едет к клиенту в определенный район.

Перед выездом он открывает dashboard и видит на карте всех клиентов поблизости. Далее он:

- обзванивает клиентов рядом;
- предлагает обслуживание или дополнительные услуги;
- добавляет дополнительные выезды в маршрут.

### Результат для бизнеса

- увеличение среднего чека;
- снижение холостых поездок;
- рост загрузки мастеров;
- дополнительная выручка без увеличения рекламного бюджета.

# Как работает система

## 1. Ввод данных мастером

Мастер заполняет информацию о клиенте в Google Sheets:

- ФИО;
- адрес;
- телефон;
- комментарий;
- дату обслуживания;
- статус и другие параметры.

После заполнения мастер устанавливает специальный флаг `ready`.

---

## 2. Автоматическая обработка данных

Python-скрипт по расписанию:

- подключается к Google Sheets;
- проверяет наличие флага `ready`;
- забирает новые строки;
- сохраняет данные в базу.

### Логика обновления

- если клиент новый — создается новая запись;
- если клиент уже существует — данные обновляются.

---

## 3. Геокодирование адресов

После загрузки данных система автоматически получает координаты адресов через сервисы Яндекса.

В базу сохраняются:

- `latitude (lat)`
- `longitude (lon)`

Это позволяет отображать клиентов на карте.

---

## 4. Визуализация в Yandex DataLens

В DataLens построен dashboard с картой клиентов.

### Реализовано

- отображение клиентов по координатам;
- карточки клиентов при наведении;
- фильтрация по городам и районам;
- фильтрация по телефону и другим параметрам;
- цветовая сегментация клиентов;
- быстрый поиск клиентов рядом.
