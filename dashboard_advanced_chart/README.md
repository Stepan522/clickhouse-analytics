## Описание проекта

Разработан кастомный Advanced Chart для Yandex DataLens в формате управленческой таблицы по performance-маркетингу.
Чарт показывает план/факт по бюджету, оформленной выручке и ДРР в разрезе рекламных каналов.

---

# Реализованный функционал

-Подключение данных из датасета DataLens
-Фильтрация по году, месяцу и неделе
-Расчет плановых значений на выбранный период
-Расчет фактических показателей
-Разделение Яндекс Директа на брендовый и небрендовый трафик
-Итоговая строка Total
-HTML-таблица с кастомной версткой внутри Advanced Chart

# Используемые формулы

## Бюджет, план:
day_total_cost * количество дней выбранного периода

## Бюджет, факт:
cost_ak * НДС

## Оформленная выручка, план:
day_revenue_plan * количество дней выбранного периода

## Оформленная выручка, факт:
revenue

## ДРР:
Бюджет / Оформленная выручка

## Особенности реализации

Плановые показатели пересчитываются динамически в зависимости от выбранного периода.
Если выбрана неделя, план считается только за дни этой недели с учетом текущей даты.

# Результат

Получился интерактивный отчет, который позволяет быстро сравнивать план и факт по ключевым маркетинговым метрикам и контролировать выполнение бюджета и ДРР.
