# ClickHouse Analytics Portfolio

Портфолио SQL и BI-кейсов, связанных с:

- ClickHouse аналитикой;
- маркетинговой атрибуцией;
- AppMetrica;
- WebMetrica;
- CRM-интеграциями;
- BI-дашбордами;
- обработкой больших объемов данных.

## Stack

- ClickHouse
- Postgres
- SQL
- Python
- AppMetrica
- Yandex Direct
- DataLens
- Power BI
