# CPA Channel Segmentation

## Задача

Получить сегментацию CPA-канала на основе подготовленного атрибуционного датасета.

Описание исходного датасета:
[AppMetrica Attribution Dataset](../sql/README.md)

## Логика сегментации

Для сегментации использовались поля:

- `parameters`
- `partner`

Маппинг источников и логика сегментации были предоставлены клиентом.

## Особенности реализации

Строка считается валидной, если:

- попадает в окно атрибуции 30 дней;
- либо содержит специальный `flag` в разметке.

Дополнительно реализована фильтрация и нормализация источников для корректного распределения трафика.

## Результат

На выходе формируется полностью готовый датасет CPA-источника:

- соответствующий ТЗ клиента;
- с корректной атрибуцией;
- без невалидных строк;
- пригодный для дальнейшей BI-аналитики и расчета маркетинговых метрик.
